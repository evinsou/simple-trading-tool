## The tool description and way of use ##
### Description ###
At first, you download a html file. Knowing where it was downloaded, locate the folder and open the html file.
You open the file like any document — double-click the HTML file, and it opens in your browser.
What you see:
* A table with data
* Checkboxes for Systems and Products
* Reset buttons respectively for Systems and Products

### How you use it: ###
* Tick one or more systems → the table instantly shows only those
* Tick some products → now you’re narrowing it further
* Click column headers → sort (e.g., cheapest price first, or most expensive by clicking one more time)
* Use the search box → type anything (e.g. a commodity name) to quickly find rows
* Click Reset → everything comes back
